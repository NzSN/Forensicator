Graph Module

Takes flat candidates + roots from scanner, builds an indexed pointer graph

PointerGraph data structure

nodes: Vec<GraphNode>
adj_out: Vec<Vec<EdgeIndex>>  // outgoing edges per node
adj_in: Vec<Vec<EdgeIndex>>   // incoming edges per node (reverse index)
edges: Vec<GraphEdge>
va_to_node: HashMap<u64, NodeIndex>
roots: Vec<NodeIndex>         // entry points for traversal

Construction

1Insert nodes — for each unique source_va and target_va in candidates, create a node. Dedup by VA.
2Insert edges — for each candidate, add edge from source node to target node, carrying confidence + evidence.
3Merge duplicates — if two candidates produce the same (source, target) edge, merge confidences (max) and union evidence. Keep best score.
4Mark roots — resolve scanner Root[] to NodeIndex via va_to_node map.

GraphNode & GraphEdge

GraphNode {
  va: u64,
  region_class: RegionClass,  // Image/Stack/Heap/Mapped/Other
  out_degree: usize, in_degree: usize,
  is_root: bool,
}

GraphEdge {
  from: NodeIndex,
  to: NodeIndex,
  confidence: f64,
  evidence: Vec<String>,
  matched_by: Vec<String>,  // pattern names
}

Public API (consumed by query module + S3)

.node(va) → Option<&GraphNode>
.node_count() → usize
.edge_count() → usize
.root_nodes() → &[NodeIndex]
.edges_from(node) → &[GraphEdge]
.edges_to(node) → &[GraphEdge]
.iter_nodes() → impl Iterator<Item = &GraphNode>

TLA+ spec

New PointerGraph.tla models: finite set of VAs as nodes, each edge carries a confidence score and must connect nodes that exist. Traversal follows edges with confidence ≥ some threshold. Invariants: no self-loops on non-heap regions, all root nodes are reachable from themselves, edge confidence in [0,1].
